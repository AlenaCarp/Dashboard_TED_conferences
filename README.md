# Дашборд TED конференций: лучшие практики для новых мероприятий  

| [Ссылка на дашборд в DataLens](https://datalens.yandex/de66exj6w5kcy?_theme=dark) |
|-----------------------------------------------------------------------------------|

Дашборд создан для анализа прошедших TED-конференций. Он помогает организаторам выбрать тематики, спикеров и форматы будущих мероприятий, чтобы они были интересными и запоминающимися.  

## Цель  
Показать популярные темы выступлений, реакцию зрителей на них, количество выступающих на конференциях, продолжительность выступлений, популярных спикеров.  

## Показатели  

  Кол-во конференций  
  Кол-во выступлений  
  Кол-во выступающих  
  Кол-во тегов  
  Топ тем выступлений и среднее кол-во просмотров  
  Топ смешных выступлений  
  Информация о конференциях  
  Популярные профессии спикеров  
  Информация о выступлениях  

## Фильтры  
Общие - Страны, Даты,Конференции, Теги  
Частные - Название, Спикер, Профессия для чарта Инфо о выступлениях  

## Источник  
Учебная база данных, предоставленная Яндекс.Практикумом (курc Аналитик данных)
